## Descripción

Serverless Star Wars API. Hecha con [NestJs](https://github.com/nestjs/nest) y [SWAPI](https://swapi.py4e.com/).

## Instalación

```bash
$ npm install
```

## Ejecutar el App

```bash
# local
$ sls offline

# deploy to AWS
$ sls deploy
```

## Test

```bash
# unit tests
$ npm run test
```
